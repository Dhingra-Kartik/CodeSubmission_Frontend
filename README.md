# 🚀 Online Judge Frontend

A modern React frontend for an Online Judge platform that allows users to browse coding problems, write solutions in multiple languages, submit code for evaluation, and receive live results through WebSockets.

## ✨ Features

- 🔐 User Authentication (JWT)
- 📝 Register & Login
- 🏠 Problem Listing Page
- 📖 Detailed Problem Page
- 💻 Monaco Code Editor
- 🌐 Multiple Language Support
  - C++
  - Java
  - Python
- ⚡ Real-time Submission Results using Socket.IO
- 📊 Submission Status Display
- 🎨 Responsive UI with CSS Modules
- 🔒 Protected Routes

- ### Frontend

- React
- React Router DOM
- Axios
- Socket.IO Client
- Monaco Editor
- CSS Modules

## 🔗 Backend Services

The frontend communicates with multiple backend microservices.

| Service | Purpose |
|----------|---------|
| Auth Service | User Authentication |
| Problem Service | Fetch Coding Problems |
| Submission Service | Submit Code |
| Socket.IO Server | Live Submission Updates |

### HOW TO USE THIS
Clone the repository

Move into the project 

Install dependencies

Create a ".env" file

Start the server


Built as part of a **Microservices-based Online Judge Platform** using React, Socket.IO, Docker, BullMQ, and Node.js.
